# Gemini UI Forge 帮助手册

欢迎使用 **Gemini UI Forge**！本工具利用 Google Gemini AI 与 Compose Multiplatform 打造，旨在通过 AI 能力简化游戏 UI 开发流程。

## 1. 核心模式介绍

### 🏠 首页 (Home)
- 查看和管理所有项目模板。
- 点击“编辑布局”进入 UI 结构定义模式。
- 点击“生成资源”进入 AI 图片产出模式。

### 📐 布局编辑模式 (Layout Design)
- **双击模块**：进入该组的“隔离编辑模式”，此时只能操作该组的直接子项，防止误触其他图层。
- **双击空白**：退出当前的隔离编辑模式。
- **拖拽与缩放**：直接通过鼠标拖动模块位置，使用滚轮或顶部工具栏缩放画布。
- **精准编辑**：在右侧属性面板直接输入 X、Y、W、H 的数值。

### 🎨 资源生成模式 (Asset Generation)
- **生图指令**：在右侧输入描述词，点击“开始生成”。
- **透明背景**：开启“透明背景”开关，系统将自动利用 Python (rembg) 进行抠图。
- **铺满模块**：新生成的图片会自动裁剪透明边框，确保完美填满你定义的 90x60 等矩形区域。

## 2. 环境配置

如果您需要使用**本地背景移除（抠图）**功能，请确保系统已安装：
- **Python 3.9+**
- **rembg**, **pillow**, **onnxruntime** 库
- 您可以在“设置 -> 环境检测”中一键检测并自动修复环境。

## 3. 快捷键
- `Ctrl + Z`: 撤销操作
- `Ctrl + Shift + Z` / `Ctrl + Y`: 重做操作
- `Ctrl + S`: 手动保存当前模板
- `Delete`: 删除选中图层
